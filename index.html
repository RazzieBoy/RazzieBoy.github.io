My name is Rasmus Nordberg

I am from a small town in the southern part of Sweden. From a young age I was fascinated by computers and video games. This fascination led to me studying video game development in high school at LBS Halmstad, directly after I graduated high school I packed my bags and went to further my knowledge and education in Software Engineering at Yoobee Colleges in New Zealand.

Overwave

A roguelike bullet hell survival game where the player has to try and survive waves of enemies with increasing difficulty as the player progresses to higher levels in the game.
This game was made in Godot with C# and it was created over an 16 week time span where 8 weeks was for planning and figuring out the game concept fully and the other 8 weeks was used for the development of the game.
Github Repo Link

Threads

A 3D platformer made in Unity (Version 2022) where the goal of the game is to get to the end of the level as quickly as possible with the existing game mechanics. It does support LAN connection multiplayer (local area network). This project was made over a 16 week time span where the first 8 weeks was used for research and planning the project and the last 8 weeks was used for the development of the game.
Video provided is an MVP (minimal viable product) showcase.
Github Repo Link

The Breakout

A VR (virtual reality) escape room created in Unreal Engine with C++ Blueprints. The goal of this project was to try and get a good understanding of what it is like to create virtual reality games. This was done by creating an escape room in which the player has to figure out how to escape as quickly as possible. This project was created over a 16 week period where the first 8 weeks was dedicated to planning the project and the last 8 weeks were used for the development proccess of the project.
This project was created with a classmate of mine at Yoobee Colleges.
