About Me

Hello, my name is Rasmus Nordberg, and I am a Software Engineer passionate about video game development and interactive experiences.

I grew up in the southern part of Sweden and discovered my passion for computers and video games at an early age. This led me to study video game development at LBS Halmstad, where I built my first projects.

After graduating high school, I pursued further studies in Software Engineering at Yoobee Colleges in New Zealand, where I expanded my skills in C++, C#, Unity, Godot, and Unreal Engine.

My focus as a is on creating engaging, polished gameplay experiences with smooth mechanics and scalable systems. I enjoy combining technical problem-solving with creative design.

Projects

Minesweeper

A simple minesweeper clone that runs in a console window, this game was built in visual studio 2022 using c++

Video provided is an MVP (minimal viable product) showcase. Github Repo Link

Overwave

A roguelike bullet hell survival game where the player has to try and survive waves of enemies with increasing difficulty as the player progresses to higher levels in the game. This game was made in Godot with C# and it was created over an 16 week time span where 8 weeks was for planning and figuring out the game concept fully and the other 8 weeks was used for the development of the game.

Video provided is an MVP (minimal viable product) showcase. Github Repo Link

Threads

A 3D platformer made in Unity (Version 2022) where the goal of the game is to get to the end of the level as quickly as possible with the existing game mechanics. It does support LAN connection multiplayer (local area network). This project was made over a 16 week time span where the first 8 weeks was used for research and planning the project and the last 8 weeks was used for the development of the game.

Video provided is an MVP (minimal viable product) showcase. Github Repo Link

The Breakout

A VR (virtual reality) escape room created in Unreal Engine with C++ Blueprints. The goal of this project was to try and get a good understanding of what it is like to create virtual reality games. This was done by creating an escape room in which the player has to figure out how to escape as quickly as possible. This project was created over a 16 week period where the first 8 weeks was dedicated to planning the project and the last 8 weeks were used for the development proccess of the project.
This project was created with a classmate of mine at Yoobee Colleges.

Video provided is an MVP (minimal viable product) showcase. Github Repo Link

Om Mig

Hej, jag heter Rasmus Nordberg och jag är en mjukvaruingenjör med passion för spelutveckling och interaktiva upplevelser.

Jag växte upp i södra Sverige och upptäckte mitt intresse för datorer och videospel i tidig ålder. Detta ledde mig till att studera spelutveckling på LBS Halmstad, här lärde jag mig grunderna i programmering och spelutveckling.

Efter att ha avslutat gymnasiet fortsatte jag mina studier inom programmering och spelutveckling på Yoobee Colleges i Nya Zeeland, där utökade jag mina färdigheter inom C++, C#, Unity, Godot och Unreal Engine.

Mitt fokus som mjukvaruingenjör ligger på att skapa engagerande, polerade spelupplevelser med smidiga mekaniker och skalbara system.

The Breakout

Ett VR (virtual reality) escape room skapat i Unreal Engine med C++ Blueprints. Målet med detta projekt var att försöka få en bra förståelse för hur det är att skapa virtuella verklighetsspel. Detta gjordes genom att skapa ett escape room där spelaren måste lista ut hur man tar sig ut så snabbt som möjligt. Detta projekt skapades över en 16-veckors period där de första 8 veckorna användes för planering av projektet och de sista 8 veckorna användes för utvecklingsprocessen av projektet.
Detta projekt skapades tillsammans med en klasskamrat på Yoobee Colleges.
